# Campanha Dove 2026 | Mirror ✦

O **Mirror** é um projeto educacional e artístico que utiliza Inteligência Artificial (processada 100% localmente no navegador) para provocar reflexões sobre padrões de beleza, identidade e a cultura dos filtros em redes sociais.

## ✨ Funcionalidades

- **Ajuste de Enquadramento**: Integração com `Cropper.js` para garantir que o rosto esteja perfeitamente centralizado.
- **SmartCrop**: Detecção automática de conteúdo para sugerir o melhor recorte.
- **Super Harmonização IA**: Pipeline de imagem que aplica efeitos lúdicos e exagerados de suavização de pele, saturação e simetria lateral.
- **Privacidade Total**: Nenhum dado ou imagem sai do computador do usuário. O processamento ocorre via Canvas API e filtros de imagem locais.
- **Interface Premium**: Design em Dark Mode com efeitos de glassmorphism e ícones modernos.

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3** (Vanilla)
- **Vanilla JavaScript** (Modular, mas compatível com o protocolo `file://`)
- **[Cropper.js](https://github.com/fengyuanchen/cropperjs)** (Recorte Profissional)
- **[SmartCrop.js](https://github.com/jwagner/smartcrop.js)** (Sugestão de enquadramento automática)
- **[Lucide Icons](https://lucide.dev/)** (Iconografia moderna)

## 🚀 Como Rodar

Basta baixar o repositório e abrir o arquivo `index.html` em qualquer navegador moderno. Não é necessário servidor ou instalação de dependências.
